<h3 align="center">
CONTROLS
</h3>
<p align="center">
Move mouse to the edges of the screen to move around<br>
Left Click: place active tile<br>
Right Click: rotate active tile<br>
F: hold active tile<br>
C: centre to world origin<br>
R: restart<br>
ESC: quit<br>
</p>
